# Controle de Eventos

Aplicacao React/Vite para controle de eventos, cadastro de itens, faturamento e geracao de contratos usando Supabase.

## Variaveis de ambiente

Crie as variaveis abaixo no ambiente local e tambem no deploy da Vercel:

```env
VITE_SUPABASE_URL=https://seu-projeto.supabase.co
VITE_SUPABASE_ANON_KEY=sua-chave-anon-publica
```

## Banco de dados

Rode o arquivo `supabase-eventos.sql` no SQL Editor do Supabase antes de usar o sistema.

## Scripts

```bash
npm install
npm run dev
npm run build
```
